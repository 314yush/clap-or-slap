# Mini-App Assets

This directory contains assets required for Base mini-app integration.

## Required Assets

### Icon
- **File**: `icon-512.png`
- **Size**: 512x512px
- **Format**: PNG with transparent background
- **Usage**: App icon, favicon

### Splash Screen
- **File**: `splash-1200.png`
- **Size**: 1200x1200px
- **Format**: PNG
- **Content**: Centered logo on dark background (#09090b)
- **Usage**: Loading splash screen

### Hero/OG Image
- **File**: `hero-1200x630.png`
- **Size**: 1200x630px
- **Format**: PNG
- **Content**: Game preview with branding
- **Usage**: OpenGraph image, social sharing

### Screenshots
Located in `screenshots/` directory:
1. `screenshot-1-landing.png` - Landing page
2. `screenshot-2-gameplay.png` - Gameplay (split-screen)
3. `screenshot-3-leaderboard.png` - Leaderboard
4. `screenshot-4-loss.png` - Loss screen with share
5. `screenshot-5-streak.png` - Streak milestone celebration

**Size**: 1200x800px each
**Format**: PNG

## Notes

All assets should maintain the dripping liquid brand aesthetic and use the brand color palette (blue → orange → pink gradient).
