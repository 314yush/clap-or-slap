# Character Illustration Specifications

## Design Requirements

### Style Guide
- **Aesthetic**: Dripping liquid, cartoonish, playful, energetic
- **Color Scheme**: Match brand colors (blue/orange/pink gradients)
- **Effect**: Glossy/shiny surfaces, liquid drips, 3D appearance
- **Consistency**: All characters should share the same art style

### Character Structure

#### Face
- **Token Logo**: Use actual token logo as the character's face
- **Size**: Prominent, centered on head
- **Style**: Can be stylized but recognizable
- **Background**: Circular or rounded square background matching token colors

#### Body
- **Type**: Humanoid/cartoon character
- **Proportions**: Slightly exaggerated (larger head, expressive)
- **Pose**: Army/military stance (see poses below)
- **Colors**: Use brand gradient colors matching the token theme

#### Poses (Choose one per character)
1. **Saluting** - Right hand raised to forehead, left at side
2. **At Attention** - Standing straight, hands at sides, feet together
3. **Battle-Ready** - Crouched slightly, fists up, ready stance
4. **Marching** - One leg forward, arms swinging
5. **Facing Off** - Side profile, one hand pointing forward

### Liquid Effects
- **Drips**: Add liquid drips from bottom of character
- **Color**: Match character's primary color
- **Style**: Rounded, organic shapes
- **Placement**: Bottom of body, arms, or head

### Glossy/Shiny Effect
- **Highlights**: White highlights on upper-left surfaces
- **Gradient**: Use gradients for depth
- **Outline**: Dark outline matching brand outline colors

## Technical Specs

### File Format
- **Format**: PNG with transparent background
- **Size**: 400-600px width (height varies by pose)
- **Resolution**: 2x for retina displays (800-1200px width)
- **Color Space**: RGB, sRGB

### Naming Convention
- `{token}-character.png` (lowercase)
- Example: `eth-character.png`, `avnt-character.png`

### Export Settings
- **Transparency**: Enabled
- **Compression**: Optimized for web
- **Quality**: High (90%+)

## Design Tools Recommendations
- Figma, Adobe Illustrator, or Procreate
- Use vector graphics for scalability
- Export at 2x resolution for web
